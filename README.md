# 🥁 Interactive Drum Kit

## 🎵 Description

A funny and dynamic web page where you can unleash your musical creativity by playing an interactive virtual drum kit.

---

## 🛠️ Technologies Used  

- **HTML** – Structure of the web page  
- **CSS** – Styling and layout
